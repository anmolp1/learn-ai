# Track: Security

Supplementary material for security professionals taking the Learn AI program.

## Focus Areas
- AI-assisted vulnerability scanning and assessment
- Automated security report generation
- Using AI for code review with a security lens
- AI-powered threat modeling and risk assessment

## Capstone Ideas
- **Vuln Scanner Report Generator:** AI scans configs/code, categorizes findings, generates executive-friendly reports
- **Security Policy Reviewer:** AI reviews security policies against compliance frameworks (SOC2, ISO 27001)
- **Threat Model Assistant:** AI helps create threat models from system architecture descriptions
- **Incident Response Playbook Generator:** AI generates response playbooks from past incident data
- **Dependency Auditor:** AI reviews project dependencies for known vulnerabilities and suggests updates

## Important Notes
- Always use test/sandbox environments for scanning exercises
- Never point AI-assisted tools at production systems without authorization
- Be transparent about AI use in security assessments
- AI output in security contexts requires extra scrutiny — false negatives are dangerous
