# Learn AI: Data Engineering Track

Build, test, deploy, and monitor data pipelines using AI as your co-pilot. This track is for data engineers and analysts who want to integrate AI tools into their daily workflow — from writing Terraform configs to debugging dbt models to automating pipeline monitoring.

## Who This Is For

**Mixed levels welcome:**

- **If you're newer to DE:** You know SQL and Python basics. You've heard of Terraform, dbt, and Airflow but haven't used them deeply. AI will help you learn these tools faster by scaffolding configs, explaining patterns, and giving you a running start on real infrastructure.

- **If you're experienced:** You already build and maintain pipelines. You know Terraform and dbt. You want AI to 10x your speed — generate boilerplate, review configs, write tests, monitor pipelines, and automate the tedious parts so you focus on architecture and data quality.

Both levels will build a real, deployable data pipeline by the end.

## What You'll Build (Capstone)

A complete, AI-assisted data pipeline that:
- Ingests data from a real source
- Transforms it with documented, tested logic
- Deploys to cloud infrastructure via Terraform
- Has monitoring and alerting
- Lives in a public GitHub repo with documentation
