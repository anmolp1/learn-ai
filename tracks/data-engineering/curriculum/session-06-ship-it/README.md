# Session 6: Ship It — Demo Day

**Format:** 90 min live
**Goal:** Present your capstone, celebrate, and plan what's next

## Session Plan

### Lightning Talks (60 min)
Each participant presents their capstone pipeline:
- **5 minutes per person** (strict)
- Format: What data problem did you solve? Show the pipeline running. What did AI help with? What did you learn?
- Live demo encouraged — show `terraform plan`, a pipeline run, monitoring in action

### Presentation Tiers
Pick the tier that matches where your project is. All tiers are valid — the rubric rewards your learning journey, not just completeness.

**Tier 1 — Full Demo** (pipeline running end-to-end):
```
1. The Problem (30 sec) — What data problem does this pipeline solve?
2. Architecture (1 min) — Data source > Ingestion > Transform > Load > Monitor
3. Live Demo (1.5 min) — Show it running. Show the output. Show the monitoring.
4. AI Integration (1 min) — What did AI write? What did you fix?
5. What I Learned (1 min) — Key takeaway for other data engineers.
```

**Tier 2 — Code Walkthrough** (components work individually, full pipeline isn't wired up):
```
1. The Problem (30 sec) — What data problem were you solving?
2. What Works (2 min) — Walk through the components that work. Show code, show test output.
3. What's Next (1 min) — What would you do to finish it? Be specific.
4. AI Integration (1 min) — Where AI helped most and where it fell short.
5. What I Learned (30 sec) — Key takeaway.
```

**Tier 3 — Learning Story** (you learned a lot even if the pipeline isn't complete):
```
1. What I Tried (1 min) — What did you set out to build?
2. What I Built (1.5 min) — Show whatever you have. Code, configs, tests, even prompts.
3. What Went Wrong (1 min) — What blocked you? What would you do differently?
4. What I Learned (1.5 min) — Key takeaways about AI-assisted development.
```

### Peer Recognition (10 min)
- Each participant names one thing they admired about someone else's project
- Share with `#de-kudos` in the community

### What's Next (15 min)
- Evolving your pipeline: what would you add with more time?
- Keeping up with AI tools for DE (what to watch, what to try)
- Advanced topics: real-time pipelines, data mesh, ML pipelines, cost optimization
- Contributing back: help improve the curriculum for the next cohort

### Closing (5 min)
- Feedback survey
- Community stays open
- Showcase: add your project to the [capstone showcase](../../capstone/showcase/)
