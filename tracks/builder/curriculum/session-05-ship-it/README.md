# Session 5: Ship It — Demo Day

**Format:** 90 min live
**Goal:** Present your app, celebrate, and plan what's next

## Session Plan

### Lightning Talks (60 min)
Each participant demos their app:
- **5 minutes per person** (strict)
- **Live demo required** — show the deployed app in a browser, not slides

### Presentation Template
```
1. The Problem (30 sec)
   What problem does your app solve? Who is it for?

2. Live Demo (2 min)
   Open the app. Walk through the main user flow.
   Show the most impressive feature.

3. How It Was Built (1 min)
   What tools did you use? What did AI build vs. what did you build?
   Show one example of a prompt that worked really well.

4. What Surprised You (30 sec)
   One thing about building with AI that you didn't expect.

5. What's Next (1 min)
   Will you keep building this? What would you add?
```

### Peer Recognition (10 min)
- Each participant highlights one thing they admired about someone else's app
- Community: `#builder-kudos`

### What's Next (15 min)
- Getting users: how to share your app and get feedback
- Iterating: using AI to add features post-bootcamp
- Learning more: frameworks, APIs, deployment, AI tools
- Building in public: sharing your journey on social media, writing about it

### Closing (5 min)
- Feedback survey
- Add your project to the [capstone showcase](../../capstone/showcase/)
- Community stays open — keep building, keep sharing
