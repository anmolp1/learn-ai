# Session 0: Pre-Work

**Format:** Async (complete before Session 1)
**Time Required:** ~60-90 minutes
**Goal:** Environment ready, baseline established, app idea chosen

## Setup

### Accounts (All Free)
1. **Claude** — [claude.ai](https://claude.ai). Test it: ask Claude to write a "Hello World" web server in your preferred language.
2. **GitHub** — [github.com](https://github.com). If new to Git, complete [GitHub Skills intro](https://skills.github.com/).
3. **Deployment platform** — Pick one and create an account:
   - [Vercel](https://vercel.com) — Best for frontend/Next.js apps
   - [Railway](https://railway.app) — Best for backend/full-stack apps
   - [Fly.io](https://fly.io) — Best for Docker-based apps
   - [Replit](https://replit.com) — Best for non-technical builders (built-in editor + hosting)

### Local Environment

**If you code:**
- Code editor: [VS Code](https://code.visualstudio.com/) or [Cursor](https://cursor.sh/)
- Node.js 18+ OR Python 3.9+ (depending on your stack)
- Git installed and configured
- Optional: [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) for terminal-based AI coding

**If you're non-technical:**
- [Cursor](https://cursor.sh/) — recommended, has the most AI built in
- Node.js 18+ (we'll use JavaScript/React for the most beginner-friendly path)
- Git installed (Cursor has built-in Git UI)
- Don't panic — AI will write most of the code. Your job is to direct it.

### Verify Everything Works
- [ ] Code editor opens and works
- [ ] `node --version` or `python --version` returns a version
- [ ] `git --version` works
- [ ] Claude responds to a prompt
- [ ] GitHub repo created for your capstone
- [ ] Deployment platform account created

## Baseline Challenge

### The Task
Build a very simple web page or script:
- A page that displays "Hello, [your name]!" with a button that changes the greeting
- OR a script that takes input and produces output (calculator, converter, formatter)

### Do It Twice
1. **Without AI:** Build it yourself. Time yourself.
2. **With AI:** Ask Claude to build it. Time yourself.

### Reflect (3-5 sentences)
- Speed difference?
- Was the AI-generated version better or worse? In what way?
- What was surprising about the experience?

Share in community with `#builder-baseline`.

## Choose Your Capstone App

Pick an app you want to build. Good projects have:
- **A clear user:** Who uses this? (Even if it's just you)
- **A specific problem:** What does it solve?
- **A tight scope:** Can you describe the core feature in one sentence?
- **Something you care about:** You'll work harder on something you actually want to exist

### Idea Starters

**For developers:**
- A CLI tool that automates something you do manually
- A dashboard that visualizes data you care about
- An API that wraps a complex workflow into a simple interface
- A browser extension that adds a feature you wish existed

**For non-technical builders:**
- A landing page / portfolio site with a twist
- A simple form-based tool (calculator, quiz, configurator)
- A personal dashboard (habits, finances, reading list)
- A small utility app (Markdown editor, timer, bookmark manager)

**AI-enhanced ideas (AI is part of the product):**
- A tool that uses Claude API to summarize/analyze/generate something specific
- A chatbot for a specific use case (customer FAQ, onboarding, decision helper)
- An app that processes user input with AI and returns structured results

## Intake Form
1. Your name and background

3. Languages/frameworks you know (if any)
4. What app do you want to build?
5. Who is it for and what problem does it solve?
