# Session 5: Ship It — Presentations and Next Steps

**Format:** 90 min live
**Prerequisites:** Sessions 1-4 complete, capstone project ready
**Goal:** Celebrate, share, and plan what's next

## Session Plan

### Lightning Talks (60 min)
Each participant presents their capstone project:
- **5 minutes per person** (strict time limit)
- Format: What problem did you solve? How did AI help? What did you learn?
- Live demo encouraged (even if it's rough)
- Q&A: 1-2 questions per presenter

### Peer Recognition (10 min)
- Each participant writes one thing they admired about someone else's project
- Share in the community space with the tag `#kudos`

### What's Next (15 min)
Discussion topics:
- How to keep learning after this program
- Communities and resources to stay connected
- How to introduce AI workflows to your team/org
- The AI landscape: what's coming, what to watch

### Closing (5 min)
- Feedback survey link
- How to stay connected (community space stays open)
- How to contribute back to the program (PRs welcome)

## Presentation Template

Participants can use this structure:

```
1. The Problem (30 seconds)
   What real-world problem did you tackle?

2. The Solution (1 minute)
   What did you build? Quick demo or screenshots.

3. How AI Helped (1.5 minutes)
   Which AI tools did you use? What worked? What didn't?

4. What I Learned (1 minute)
   Key takeaway. What would you do differently?

5. What's Next (1 minute)
   How will you keep using AI in your work?
```

## Post-Program

### For Participants
- Keep your capstone repo public — it's a portfolio piece
- Continue iterating on your project
- Help the next cohort — mentor, present, contribute content

### For Instructors
- Collect feedback surveys
- Add standout capstone projects to the showcase
- Write a cohort retrospective
- Update curriculum based on what worked / what didn't
