# Session 0: Pre-Work

**Format:** Async (complete before Session 1)
**Time Required:** ~60-90 minutes
**Goal:** Get everyone set up and establish a baseline

## Objectives

By the end of this pre-work, you should:
- Have accounts and tools set up for the program
- Understand your own current AI usage level
- Have a project idea in mind for your capstone

## Setup Guide

### Required Accounts (Free Tier Is Fine)

1. **Claude** — [claude.ai](https://claude.ai)
   - Create an account if you don't have one
   - Familiarize yourself with the interface — try asking it something related to your work

2. **GitHub** — [github.com](https://github.com)
   - Create an account if you don't have one
   - If you're new to Git, complete this 15-min tutorial: [GitHub Skills](https://skills.github.com/)

3. **Code Editor** (recommended, not required for all tracks)
   - [VS Code](https://code.visualstudio.com/) — free, widely used
   - [Cursor](https://cursor.sh/) — VS Code fork with built-in AI features

### Optional (Track-Specific)
- **Data Engineering:** Google Cloud account (free tier), Terraform installed
- **Security:** A test environment or VM for scanning exercises
- **Builder:** Node.js or Python installed locally
- **Marketing:** Access to a social media management tool (Buffer, Hootsuite, etc.)

## Baseline Challenge

Before you start learning new techniques, let's see where you are now.

### The Task

Pick ONE of the following (choose the one closest to your track):

- **Data:** Write a SQL query or data transformation script for a task you do regularly
- **Security:** Write a checklist for reviewing a system's security posture
- **Builder:** Build a simple feature (a form, an API endpoint, a component)
- **Marketing:** Write a social media post, email, or blog outline for a real campaign

### Do It Twice

1. **Without AI:** Complete the task using your normal workflow. Time yourself.
2. **With AI:** Complete the same task using Claude or another AI tool. Time yourself.

### Document the Difference

Write a short reflection (3-5 sentences) covering:
- How long did each approach take?
- Where did AI help? Where did it get in the way?
- What surprised you?

Share your reflection in the community space (Discord/Slack) with the tag `#baseline`.

## Intake Form

Before Session 1, fill out this brief form (your instructor will share the link):

1. What's your name and role?
2. What industry/domain do you work in?
3. Rate your current AI tool usage (1-5, where 1 = "never used" and 5 = "use daily")
4. What do you hope to build or accomplish during this program?
5. What's your biggest question or concern about using AI at work?
