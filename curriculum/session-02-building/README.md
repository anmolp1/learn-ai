# Session 2: Building With AI — Code, Pipelines, and Automation

**Format:** 90 min live + async challenge
**Prerequisites:** Session 1 complete
**Goal:** Build something real using AI as your co-pilot

## Session Plan

### Concept (20 min): The Scaffold-Iterate-Refine Workflow
- Use AI to generate a first version quickly (scaffold)
- Review, test, and identify gaps (iterate)
- Refine with targeted prompts until it's production-ready (refine)
- Version control everything — AI output is a starting point, not a commit

### Demo (30 min): Build a Project From Scratch
Pick one based on your audience. Build it live.

**Option A — Data Engineering:**
- Use Claude to generate a Terraform config for a GCS bucket + BigQuery dataset
- Have AI write a Python data pipeline that loads, transforms, and validates data
- Push to GitHub with proper README

**Option B — Builder:**
- Use Claude to scaffold a full-stack web app (e.g., a task tracker API)
- Generate the backend, frontend, and tests in sequence
- Show how to debug AI-generated code when it doesn't work

**Option C — Marketing:**
- Use Claude to build a content calendar for a month
- Generate 3 variations of a blog post with different tones
- Create a simple automation script that formats content for multiple platforms

### Practice (25 min)
- Participants start building their capstone project with AI
- Goal: have a working skeleton pushed to a GitHub repo by end of session

### Debrief (15 min)
- Share repos in the community space
- Common struggles and solutions

## Async Challenge
- Continue building your capstone project
- Use AI to write at least one complete component/module/workflow
- Push your progress to GitHub
- Write a brief update in the community space: what you built, what AI helped with, what you had to do manually

## Key Concepts to Cover
- How to review AI-generated code (the "read before you run" principle)
- Using GitHub effectively: commits, branches, README files
- When to ask AI to generate vs. when to write it yourself
- Testing AI output: run it, break it, fix it
